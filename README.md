AI Beauty CRM

An ADHD-friendly AI-powered CRM for beauty clinics and skin therapists.

Vision

AI Beauty CRM is an open-source project focused on building a modern, low-distraction, local-first CRM system specifically designed for beauty professionals, solo therapists, and skincare clinics.

The goal is to reduce mental overload, simplify follow-ups, and create an intelligent workflow system optimized for ADHD-friendly productivity.

⸻

Planned Features

* Client profiles
* Treatment history
* Smart reminders
* AI-powered follow-up suggestions
* WhatsApp integration
* Persian RTL support
* Offline-first architecture
* Voice notes
* AI note summarization
* Fast low-distraction UI
* Tagging and filtering system
* Future multi-user support

⸻

Tech Direction

Planned stack:

* React / Next.js
* TypeScript
* Local-first database
* PWA support
* AI integrations (Claude / OpenAI / Gemini)
* Modular architecture

⸻

Status

Early open-source MVP currently in development.

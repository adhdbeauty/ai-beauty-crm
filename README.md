AI Beauty CRM

An ADHD-friendly AI-powered CRM for beauty clinics and skin therapists.

Vision

Create a low-distraction, intelligent, and local-first CRM system designed specifically for beauty professionals.

Planned Features

* Client profiles
* Treatment history
* Smart reminders
* AI follow-up system
* WhatsApp integration
* Persian RTL support
* Voice notes
* Offline-first support
* AI note summarization
* Fast ADHD-friendly UI

Status

Early open-source MVP in development.

Goals

* Help solo therapists stay organized
* Reduce mental overload
* Improve client follow-up
* Build a modern AI workflow for beauty clinics
